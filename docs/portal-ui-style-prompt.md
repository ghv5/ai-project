# 门户 UI 风格样式提示词（固定版）

> 适用范围：`car-portal-public` 门户前端（首页、案例库、登录页、顶部导航及后续新增门户页面）。
>  
> 使用原则：后续所有门户页面视觉实现，默认以本文档为唯一风格基线；若需偏离，必须先更新本文档再改代码。

## 1. 页面风格样式规范提示词

整体视觉定位：现代科技、专业政务、数据交易平台风格。采用极简主义设计，强调呼吸感、模块化布局与高度的工业标准化质感。  

核心色彩规范：  
- 主色（Primary）：`#0066FF`（科技蓝，用于高亮、主按钮、页眉激活态）  
- 辅助色（Secondary）：`#00C2FF`（天蓝色，用于渐变装饰及图标点缀）  
- 背景色（Background）：`#F5F7FA`（浅灰色底，区分白底卡片）  
- 文字色（Typography）：标题/正文 `#222222`（深黑）；辅助/说明文字 `#666666`（灰）  

排版系统：  
- 字体家族：`Sans-serif, PingFang SC, Microsoft YaHei, system-ui`  
- 标题（Headings）：`32px/28px`，`600`，字间距略微拉伸  
- 正文（Body）：`14px/16px`，`400`，行高 `1.6`  

图形与组件质感：  
- 卡片：`#FFFFFF`，圆角 `12px`  
- 阴影：`0 4px 20px rgba(0, 0, 0, 0.05)`，无明显边框  
- 按钮：圆角 `4px`，纯色 `#0066FF` 或渐变 `#0066FF -> #00C2FF`，文字白色  
- 图标：微拟态/2.5D 风格，蓝色渐变玻璃质感或蓝白线面结合  

布局与间距：  
- 版心宽度：`1200px - 1400px`（门户当前建议 `1320px`）  
- 导航栏：顶部白底通栏，高度 `64px`，右侧功能组与左侧 Logo 对称  
- Banner：全宽浅蓝科技背景，抽象线条 + 透明浮动几何体  
- 内容区：多栏对称分布（例如左 `3/4` + 右 `1/4`），模块间距 `24px - 32px`  
- 业务流程：单行线性排列，步进式虚线箭头（Dotted Arrow）连接，强调横向引导  

视觉氛围：高亮度、低饱和度背景、强逻辑感、清爽通透。运用大量白色空间（Negative Space）营造高端商业界面质感，光影效果温润，无生硬转折。

## 2. 执行约束（落地规则）

- 门户页面禁止回退到深色重背景主视觉（仅允许局部深色点缀，不可反客为主）。
- 卡片体系统一使用白底 + 12px 圆角 + 轻阴影，不允许混用多套圆角/阴影标准。
- 顶部导航与登录入口保持同一主色语义：激活态、主按钮、关键链接统一使用 `#0066FF`。
- 新增门户模块应优先复用现有组件样式层级（标题区、筛选区、卡片区、操作按钮区）。
- 任何页面视觉调整，需先校验与本文档的色彩、排版、间距、组件规范一致。
