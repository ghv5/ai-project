### Acceptance Checklist (Manual)

- 阶段模板：新建阶段可添加/删除/排序字段，至少覆盖 text/select/attachment
- 部门配置：阶段可排序；rules_avg 阶段可绑定 2+ 规则
- 评估入口校验：部门无阶段配置时，开始评估按钮禁用且 API 不创建评估记录
- rules_avg 计算：录入 2+ 规则得分后自动计算阶段得分为均值（四舍五入）
- 边界：阶段得分与规则得分被约束在 0..100
- 完成条件：全部阶段都有得分后评估自动完成，生成报告按钮可用
